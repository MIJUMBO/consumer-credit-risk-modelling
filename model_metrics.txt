1. DATA
------------------------------------------------------------------------------
Borrower data (Layer 1)  : LendingClub accepted loans 2007–2018Q4
Raw                      : 2,260,701 loans x 151 columns
Resolved (modelled)      : 1,345,350 loans  (Fully Paid / Charged Off / Default)
Overall default rate     : 19.96%  (~1:4 class imbalance)
Dev sample               : 100,000 loans (stratified on issue_year x default_flag)
Production sample        : 500,000 loans
Macro panel (Layer 2)    : 318 months x 6 cols, Jan 2000–Jun 2026, zero nulls
                           (BoC overnight rate V39079; StatCan 14-10-0287,
                            18-10-0004, 11-10-0065, insolvencies v41690973)
Architecture rule        : macro panel NEVER row-joined to loans (scenario use only)

2. FEATURES  (15 origination-time predictors, one engineer() function)
------------------------------------------------------------------------------
Affordability   : loan_to_income, dti, loan_amnt, log_annual_inc (log + p99 cap)
Credit quality  : fico_orig, revol_util (median-imputed), open_acc_band (ordinal)
Derogatory      : delinq_flag, pub_rec_flag
Employment      : employment_years, short_tenure_flag (<1yr)
Loan/vintage    : purpose_risk_tier, home_ownership, vintage_year, vintage_quarter
Vintage audit   : 39 CONSISTENT / 4 PARTIAL / 89 SPARSE -> 32 candidates -> 15 used
Encoding        : OrdinalEncoder (3 categoricals), fit on TRAIN only, unknown = -1
Standardisation : StandardScaler for Logistic Regression only (trees use raw values)
Excluded (EDA)  : grade, sub_grade, int_rate  (circular — lender's own risk pricing)
Leakage dropped : 22 post-outcome columns

10. IFRS 9 ECL FRAMEWORK  (ECL = PD x LGD x EAD)
------------------------------------------------------------------------------
LGD                  : 45% (unsecured consumer)        EAD : loan amount
SICR threshold       : calibrated PD > 0.3064 (80th pct baseline) -> Stage 2
Stage 3              : realised default, PD = 100%
Lifetime multiplier  : 2.5x 12-month PD (term mix 78.5% 36mo / 21.5% 60mo;
                       blended range 2.32–2.59)
PD–unemployment elasticity : +10% relative PD per +1pp unemployment
Reusability gate     : stage_and_ecl() shared across all scenarios; zero-shock
                       reproduces baseline to the cent
